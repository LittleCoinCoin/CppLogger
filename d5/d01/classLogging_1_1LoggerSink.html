logFunc_pf Logging::LoggerSink::loggingFunctions[LogLevel_Count] = { nullptr }
protected

The array of function pointers to log messages for each log level.

The index of the function in the array corresponds to the Logging::LogLevel. By default it is full of nullptrs. If you do not set a function pointers for a log level, the message will not be logged.
